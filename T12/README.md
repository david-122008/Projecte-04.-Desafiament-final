# T12: Fonaments del Disseny Web Comercial – Landing Page + Procés de Checkout

## Breu descripció
Aquesta tasca consisteix en una **classe magistral** per explicar l’estructura de les pàgines web comercials, centrant-se en dos pilars essencials de qualsevol botiga online:  
- **Landing Page**  
- **Procés de Checkout**

---

## Objectiu
Abans de començar a dissenyar la maqueta completa de l’e-commerce del client d’EverPia, cal entendre:  
- Com es construeix una **Landing Page** efectiva.  
- Com es defineix un **Procés de Checkout** òptim.  
- El recorregut complet que fa l’usuari, des que arriba al web fins que finalitza la compra.  

---

## Contingut de la tasca
- **Landing Page:**  
  - Funció principal: captar l’atenció i convertir visitants en clients.  
  - Elements clau: titular atractiu, CTA (Call to Action), beneficis clars, prova social, disseny visual coherent.  
  - Exemple: [Landing Page quilomètrica – Romuald Fons](https://romualdfons.com/monetizacion-webs/)  

- **Procés de Checkout:**  
  - Funció principal: facilitar la compra de manera ràpida i segura.  
  - Elements clau: formulari senzill, opcions de pagament clares, confirmació visual, reducció de friccions.  
  - Exemple: pantalles de checkout (referència visual).
